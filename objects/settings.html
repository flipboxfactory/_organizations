Settings

The Organization Settings model supports all Model operations. In addition, the following can be utilized:

# Public Properties

All of the standard Model public properties are available. In addition, the following properties are also available:

PropertyTypeDescription
defaultStatestring, nullThe organization's default state

# Public Methods

All of the standard Model public methods are available. In addition, the following methods are also available:

Returns: [integer]

Returns: boolean

#getStates()

Returns: string[]

Returns: boolean

ArgumentAcceptsDescription
